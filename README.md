# 🔑 Gebruik als "Dungeon Master"
- **Voor vertrek** → gebruik de checklist om de koffer in te pakken.  
- **Op locatie** → richt de ruimte in als je “speeltafel” (whiteboards, canvassen, props).  
- **Tijdens sessies** → gebruik props als spelmechanieken (stemmen, timers, dobbelstenen) om focus, energie en plezier te sturen.  
- **Na afloop** → documenteer resultaten met foto’s en verzamel feedbackkaarten. Alles weer terug in organizers → koffer dicht → klaar voor de volgende workshop.
